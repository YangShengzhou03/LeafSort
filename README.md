# LeafView - 您的专属相册管家

LeafView是一款高效的本地相册管理工具，集图片存储、筛选、分类、美化、分享与播放等多场景为一体，重新定义了相册的存储、管理、分享、观看方式。

## 产品定位
LeafView不是给设计师用的素材整理管理软件，而是面向普通大众用户的本地NAS或媒体照片管理软件，作为您相册的专属管家，提供便捷、智能、安全的照片和视频管理体验。

## 核心功能

### 智能分类与识别
- **场景识别**：支持狗、江河、聚餐等52种常见场景识别
- **人物识别**：自动识别并分类照片中的人物
- **地点识别**：基于照片GPS信息自动按地点分类
- **时间轴**：按拍摄时间自动组织照片和视频

### 智能影集生成
- **故事影集**：自动为纪念日、旅行等特殊日子生成精美影集
- **自定义影集**：用户可手动创建和管理相册
- **最近添加**：实时展示最新导入的照片和视频
- **我的收藏**：快速访问标记为收藏的照片

### 相似照片管理
- **相似图片查找**：智能识别相似照片并给出相似度打分
- **一键清理**：快速清理表情不佳、面部模糊、虚影等废照片
- **重复照片检测**：自动识别并删除重复照片，释放存储空间

### 智能编辑功能
- **基础编辑**：调整亮度、对比度和色彩等参数
- **批量编辑**：支持对多张照片进行批量处理
- **滤镜效果**：提供多种预设滤镜，快速美化照片

### 快速搜索
- **关键词搜索**：通过关键词快速找到需要的照片
- **自然语言搜索**：支持"去年夏天的海边照片"等自然语言查询
- **多条件筛选**：支持按时间、地点、人物、场景等多维度筛选

### 多设备支持
- **跨终端同步**：支持手机、平板及电脑多端登录
- **多设备连接**：可与电视、投影和车载屏幕等设备连接
- **备份与恢复**：支持相册照片或视频的备份和恢复

### 数据安全
- **本地存储**：所有照片和视频存储在本地，保障隐私安全
- **智能备份**：支持自动备份照片和视频
- **文件灾备**：提供文件灾备功能，防止数据丢失

## 技术特点

- **本地AI技术**：基于大模型AI技术实现相册智能化管理
- **响应式设计**：适配不同屏幕尺寸，确保在各种设备上都有良好体验
- **多语言支持**：提供简体中文与英文版本
- **轻量级**：占用资源少，运行流畅

## 适用人群

普通家庭用户、摄影爱好者、需要管理大量照片和视频的个人用户。

## 技术栈

- **前端框架**：Vue.js 3 (Composition API)
- **桌面应用框架**：Electron
- **UI组件库**：Element Plus
- **状态管理**：Vue Router
- **构建工具**：Vue CLI, electron-builder

## 安装与运行

### 开发环境

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run serve

# 启动Electron开发环境
npm run electron:dev
```

### 构建发布

```bash
# 清理构建文件
npm run clean

# 构建应用
npm run build
```
